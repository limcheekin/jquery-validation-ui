JQuery Validation UI Plugin - Client Side Validation without writing JavaScript

(Quick Reference)

resources

This tag is deprecated in favor of the resources plugin module definitions.

This tag pulls in the resources needed such as JQuery Validation library and qTip library. You can pulls in javascript and CSS resources separately by specify the type attribute such as <jqvalui:resources type="js" /> and <jqvalui:resources type="css" />.

Sample

<jqvalui:resources />
